# L Function

[TOC]

## Define

> An $L$-function is a complex analytic object that encodes arithmetic data through Euler products, Dirichlet series, and functional equations.

An **$L$-function** is a complex-valued function built from arithmetic data and expected to satisfy analytic continuation, a functional equation, and an Euler product in an appropriate region. $L$-functions unify objects from analytic number theory, automorphic forms, and arithmetic geometry.

## Properties

### Standard Features

Typical $L$-functions are first presented by a Dirichlet series and an Euler product, then extended analytically beyond their initial half-plane of convergence.

### Arithmetic Role

They connect primes, automorphic forms, and Galois representations inside a common framework.
