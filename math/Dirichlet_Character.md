# Dirichlet Character

[TOC]

## Define

> A Dirichlet character is a periodic multiplicative arithmetic function modulo an integer.

For a positive integer $q$, a **Dirichlet character modulo $q$** is a function
$$
\chi : \mathbb Z 	o \mathbb C
$$
that is periodic modulo $q$, completely multiplicative, and factors through the unit group $(\mathbb Z / q \mathbb Z)^	imes$, with the standard extension by $0$ on integers not coprime to $q$.

## Properties

### Basic Properties

A Dirichlet character satisfies
$$
\chi(mn)=\chi(m)\chi(n),\qquad \chi(n+q)=\chi(n).
$$

### Arithmetic Role

Dirichlet characters are the basic coefficients used to form Dirichlet $L$-functions and to study primes in arithmetic progressions.
