# Point Process

[TOC]

## Define

> A point process is a random locally finite configuration of points.

A **point process** is a random mechanism that produces a countable set of points in a space, or equivalently a random locally finite counting measure
$$
N = \sum_i \delta_{X_i}.
$$
For each measurable region $A$, the random variable $N(A)$ counts the number of points falling in $A$.

## Properties

### Counting-Measure Viewpoint

A point process is often studied through the family of counting variables $N(A)$.

### Intensity

The intensity describes the expected local density of points.
