# Number Field

[TOC]

## Define

> A number field is a finite field extension of the rational numbers.

A **number field** is a field $K$ such that
$$
[K : \mathbb Q] < \infty.
$$
Number fields are the basic global objects of algebraic number theory.

## Properties

### Ring of Integers

Each number field has a distinguished ring of integers, which plays the role of an arithmetic lattice inside the field.

### Arithmetic Role

Prime decomposition, ramification, class groups, and unit groups are studied inside number fields.
